Meet the MAS Library

Designed for developers and non-developers alike, MAS allows you to design complex multi-agent systems with minimal code.

Structured Agents

Create agents that use tools in very specific ways, ensuring reliability and control.

Provider Independent

Seamlessly integrate with OpenAI, Google, Groq, Anthropic, DeepSeek, and local models.

JSON Configuration

Define your entire system in JSON. Readable by humans, writable by LLMs.

Auto-Generation

Describe your system in plain English, and let MAS generate the configuration for you.

Core Capabilities

Native Bot Support

Out-of-the-box support for WhatsApp and Telegram bots. Deploy your agents to where your users are.

Multimodal

Handle text, images, and audio natively. The block-based system ensures seamless data exchange.

Agentic Workflows

Design complex flows with Agents, Tools, Processes, and Automations. Support for branching, loops, and more.

Centralized History

Robust message history management with SQLite, supporting multi-user sessions and context-aware agents.

Custom Tools

Easily extend agent capabilities with custom Python functions defined as tools.

Long-term Memory

Built-in support for vector databases to give your agents persistent context and knowledge.

Role-Playing

Define distinct personas and roles for each agent to specialize tasks and interactions.

Async Operations

Run heavy tasks in the background without blocking the main conversation flow.

Observability

Track agent performance, token usage, and costs with built-in logging and metrics.

MAWS Extension

MAS on AWS

Deploy your MAS bots to AWS Lambda + API Gateway with almost no effort.

  • Lazy initialization for fast cold starts
  • Webhook handling for WhatsApp & Telegram
  • Background processing via self-invocation
  • DynamoDB locking & S3 synchronization

Simple & Powerful

Install via pip install --upgrade git+https://github.com/mneuronico/multi-agent-system-library

from mas import AgentSystemManager

# Initialize the manager
manager = AgentSystemManager(config="config.json") 
output = manager.run(input="Hello world!", verbose=True)

print(output)
